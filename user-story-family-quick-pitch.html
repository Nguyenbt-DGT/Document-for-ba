As a Manulife agent using M-Smart,
I want to describe my customer's full family in a single natural language message,
so that M-Smart recommends a complete family insurance package — with the right base product for the main insured and appropriate riders for the spouse and children — all within the stated budget, and produces a single illustration PDF ready to present to the customer.
Background

Agents frequently sell family insurance packages where one policy covers the main breadwinner as the base insured, with the spouse and children protected through riders on the same policy. Today, M-Smart only supports a single person per quick pitch, forcing agents to manually calculate each family member separately in ePOS — increasing errors, wasting time, and missing the chance to present a compelling whole-family proposal. This enhancement lets the agent describe the family in one message and receive a complete, budget-aware, pitch-ready family package instantly.
